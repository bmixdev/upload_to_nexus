# Nexus Raw Uploader (Win11, BAT + HTA)

## Назначение

Скрипт [`run_uploader.bat`](run_uploader.bat) открывает форму [`nexus_uploader.hta`](nexus_uploader.hta), где можно:

- указать URL raw-репозитория Nexus;
- указать подпуть внутри репозитория;
- выбрать локальный файл через кнопку выбора;
- ввести логин и пароль;
- выполнить последовательность **DELETE старого файла** → **UPLOAD нового файла**;
- видеть лог выполнения в memo-поле.

## Состав

- [`run_uploader.bat`](run_uploader.bat) — запуск формы через `mshta.exe`.
- [`nexus_uploader.hta`](nexus_uploader.hta) — UI + JS-логика + вызовы PowerShell-команд.

## Как использовать

1. Поместите [`run_uploader.bat`](run_uploader.bat) и [`nexus_uploader.hta`](nexus_uploader.hta) в одну папку.
2. Запустите [`run_uploader.bat`](run_uploader.bat) двойным кликом.
3. В открывшейся форме заполните поля:
   - **URL репозитория** — например `http://localhost:8081/repository/my-raw`
   - **Подпуть в репозитории** — например `releases/app` (можно пусто)
   - **Локальный файл** — выберите через кнопку **Выбрать...**
   - **Пользователь Nexus**
   - **Пароль Nexus**
4. Нажмите **Удалить и загрузить**.
5. Контролируйте статус в поле **Лог (memo)**.

## Правила обработки ответов

На этапе DELETE:

- `2xx` — удаление успешно, переход к загрузке;
- `403` или `404` — логируется предупреждение и загрузка продолжается;
- любой другой код — операция останавливается.

На этапе UPLOAD:

- `2xx` — загрузка завершена успешно;
- любой другой код — вывод ошибки в логе.

## Как собирается целевой URL

Финальный URL артефакта:

`{repoUrl}/{repoPath}/{fileName}`

где:

- `repoUrl` — базовый URL из формы (без конечного `/`);
- `repoPath` — подпуть (без начальных/конечных `/`, может быть пустым);
- `fileName` — имя выбранного локального файла.

Если `repoPath` пуст, получится:

`{repoUrl}/{fileName}`

## Диагностика типовых проблем

- Ошибка `403` на DELETE: обычно недостаточно прав на удаление, но загрузка продолжится по заданному правилу.
- Ошибка `401`/`403` на UPLOAD: проверьте логин/пароль и роли пользователя в Nexus.
- Ошибка «Файл не найден»: проверьте фактический путь, выбранный в форме.
- Ошибки TLS/сертификата: проверьте доступ к Nexus из PowerShell на этой машине.

## Ограничения

- Решение ориентировано на Windows 11 и наличие `PowerShell` + `mshta.exe`.
- Форма использует HTA/ActiveX-подход, который подходит для локального админского сценария.
